finalapp is ready to use apk file. It can be installed on the Android Device and used directly.
1. Copy file to Android device
2. Allow install from unknown 
3. Install and use

croponlocation folder is a android project which contains files and codes required to develop
the app. One can open the project in Android Studio and get the layouts and codes which are used in
the ap.

1. Open Android Studio
2. On top Go to File->Open->Locate the folder croplocation->select.


Server_crop is the server which takes the post request from the app and returns the recommendation.
It is a python-flask server which is running 24 hours on Heroku.
